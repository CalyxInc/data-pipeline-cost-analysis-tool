## Why

## What Changes

- **BREAKING**：以全新設計完整替換 `index.html` 的 HTML / CSS
- 計算邏輯（JS 函式）維持不變，僅重構視覺層
- 介面風格升級：明確的視覺層次、數據儀表板感、成本項目更易對比閱讀
